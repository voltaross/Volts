Requester Pro v2.0.4

The ultimate Spotify & Twitch integration for streamers.

Download Latest Release

Includes Auto-Update Support for Windows 10/11

1 Setup Spotify API

Go to the Spotify Developer Dashboard and create a new App.

  • Click Edit Settings.
  • Add http://127.0.0.1:8888/callback to the Redirect URIs.
  • Save and copy your Client ID and Client Secret.

2 Get Twitch Token

Visit Twitch Token Generator.

Select User Access Token and ensure you check the following scopes:

channel:read:redemptions chat:read chat:edit

3 Create the Reward

In your Twitch Dashboard, create a Reward named exactly Song Request.

IMPORTANT: You must toggle "Require Viewer to Enter Text" to ON. This allows users to paste Spotify links or song names.

4 Launch & Link

Open RequesterPro.exe and paste your keys. Run the setup in order:

  1. Save Settings: Stores your API keys locally.
  2. Connect Twitch: Joins your chat and listens for rewards.
  3. Link Spotify: Controls your music playback.

5 Seamless Updates

How Updates Work

Starting with v2.0.3, you no longer need to manually download new versions. When an update is released, the app will:

  • Notify you in the app log.
  • Give you a choice to Install Now, Later, or Skip that specific version.
  • Announce the current song playing directly to your Twitch chat!

6 Add to OBS

Add a new Browser Source in OBS:

  • URL: http://127.0.0.1:8888/overlay
  • Dimensions: 400x150 (Customizable)

Requester Pro is an independent tool. All keys and data are stored locally on your machine.
