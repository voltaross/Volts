Requester v2.0

The ultimate Spotify & Twitch integration for streamers.

Download for Windows (.zip)

Supports Windows 10/11 x64

1 Setup Spotify API

Go to the Spotify Developer Dashboard and create a new App.

  • Click Edit Settings.
  • Add http://127.0.0.1:8888/callback to the Redirect URIs.
  • Save and copy your Client ID and Client Secret.

2 Get Twitch Token

Visit Twitch Token Generator.

Select User Access Token and ensure you check the following scopes:

channel:read:redemptions channel:manage:redemptions

3 Create the Reward

In your Twitch Creator Dashboard, create a new Channel Point Reward named exactly Song Request.

IMPORTANT: You must toggle "Require Viewer to Enter Text" to ON. This is how the app knows which song to play!

4 Link & Launch

Open Requester.exe and paste your keys. Follow the 1-2-3 button order:

  1. Save Settings: Locks in your keys.
  2. Connect Twitch: Starts the listener.
  3. Link Spotify: Opens your browser to authorize music control.

5 Add to OBS

Add a new Browser Source in OBS with the following settings:

  • URL: http://127.0.0.1:8888/overlay
  • Width: 400
  • Height: 150

Privacy: All keys are stored locally on your machine. No data is sent to our servers.
