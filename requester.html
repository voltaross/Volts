Requester Pro v2.0.6

Automatic Spotify & Twitch integration. Simple setup.

Download v2.0.6

1 Setup Spotify API

Create an app in the Spotify Dev Dashboard & copy both Secret ID and Client ID, paste these in the app. Set Redirect URI to http://127.0.0.1:8888/callback.

2 Launch & Link

Paste keys in Requester Pro, click Authorize, and link your accounts.

3 Auto-Announce

The app automatically sends a "Now Playing" message to Twitch chat for every new song.

4 Managing Updates

Choose to Download, Skip, or Install Later for new releases.

5 OBS Overlay

Add http://127.0.0.1:8888/overlay as a Browser Source in OBS (400x150).
